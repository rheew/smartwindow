# 창문 보안 시스템 with 라즈베리파이

# 제작 목적
- 사회적 약자로 분류 되는 거주자들을 위한 창문 보안 시스템을 구축하는 프로젝트입니다. 
  모바일 어플리케이션으로 거주지의 창문상태, 창문 제어, 실시간 영상정보, 상황 알림 서비스를 통해 안전한 생활을 도와줍니다.
  
# 주요 기능
- 실시간 창문 상태 확인
- 실시간 창문 앞 영상 확인
- 창문 원격 제어
- 자동 비 감지
- 자동 사람 감지
- 실시간 이슈 알림 서비스

# 기술 스택

- back end : Fire Base, Python, Java (Android)
- front end : Android
